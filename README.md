# CATchain

This repository provides the data and code for the CATchain-R simulation using New York City (NYC) transportation carbon emission data.

- **Vensim PLE**: Software used for the simulation (to open the `.mdl` file).
- **Data source**: NYC climate dashboard  
  https://comptroller.nyc.gov/services/for-the-public/nyc-climate-dashboard/emissions/
